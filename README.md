## 📋 About

**Wallet Live** is a web application built in Rust using the `axum` framework, `sqlx` for database interactions, and `askama` for templating. It serves as a practical project for the Santander 2026 Rust AI Developer Bootcamp, aimed at teaching Rust concepts and web development in a real-world scenario.

## 🔋 Features

### ⚙️ **Core Tech Stack**
- **Rust & Axum:** Fast and reliable web server
- **SQLx & PostgreSQL:** Asynchronous, compile-time checked database access
- **Askama:** Type-safe, compiled HTML templates
- **Tokio:** Asynchronous runtime for Rust

### 🚀 **Functionality**
- Web interface for managing digital wallets
- Secure authentication and JWT sessions
- Database migrations support
